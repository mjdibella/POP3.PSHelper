Connect-POP3: connect to POP3 server, returns session variable
	-server mandatory, server to connect to	
	-username mandatory
	-password mandatory
	-port optional
	-enableSSL optional boolean switch

Get-POP3Message: retreive all messages in mailbox or only specified numbered message
	-session mandatory, pop3 session variable
	-index optional, message number to retreive

Get-POP3MessageHeader: retreive headers for all messages in mailbox or only specified numbered message
	-session mandatory, pop3 session variable
	-index optional, message number to

Get-POP3RawMessage: retreive all messages in mailbox as unparsed string or only specified numbered message
	-session mandatory, pop3 session variable
	-index optional, message number to retreive

Remove-POP3Message: delete specified numbered message from mailbox
	-session mandatory, pop3 session variable
	-index mandatory, message number to retreive

Clear-POP3Mailbox: delete all messages in mailbox
	-session mandatory, pop3 session variable


Get-POP3Capabilites: retreive the mailbox capabilities list
	-session mandatory, pop3 session variable

Get-POP3UIDL: retreive the mailbox message list
	-session mandatory, pop3 session variable

Disconnect-POP3: disconnect from mailbox and end session
	-session mandatory, pop3 session variable
